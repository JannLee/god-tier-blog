# 블로그를 시작하며

안녕하세요! 이 블로그는 **Next.js**와 **GitHub Pages**를 사용해 만들었습니다.

## 왜 블로그를 만들었나요?

개발하면서 배운 것들을 정리하고 공유하고 싶었습니다. 노션이나 티스토리 같은 플랫폼도 좋지만, 직접 만든 블로그는 자유도가 다릅니다.

## 기술 스택

- **프레임워크**: Next.js 16 (App Router)
- **스타일링**: Tailwind CSS v4
- **콘텐츠**: MDX
- **배포**: GitHub Pages (정적 내보내기)

## 앞으로 다룰 주제들

- TypeScript 실전 패턴
- React Server Components 깊게 이해하기
- Next.js App Router 마이그레이션 경험기

앞으로 자주 업데이트할 예정이니 많은 관심 부탁드립니다!
